lava-land.ru

Авторы проекта: Бубнов Демьян, Наумов Лаврентий

Замысел.
Основной идеей проекта является возможность удаленного хранения файлов. Задача проекта дать возможность загружать и скачивать файлы, а также делиться доступом к ним с другими пользователями. Из известных аналогов есть следующие сервисы: Google Drive и Яндекс.Диск.

Функциональность.
Безусловно пользователь может загружать, скачивать файлами, делиться ими со всем интернетом или только с доверенными пользователями.
Чтобы не потеряться среди своих файлов, пользователь имеет возможность отслеживать файлы по дате загрузке, а так же может создавать и просматривать описания к файлам. Поделиться ссылкой к файлу, попросить доступ или просто поговорить можно, не выходя с сайта, во встроенном мессенджере.

Безопасность.
Доступ к сайту осуществляется по протоколу http, но идёт переход на https для обеспечения приватности пользователей. Удаленный доступ к серверу ограничен web версией и входом по ssh-ключу что исключает возможность несанкционированного доступа к ресурсу. Все поля (в том числе и названия файлов) содержат ограничения на содержание, исключающие возможность доступа к выполняемому коду и базе данных. А все файлы на сервере хранятся в обезличенном виде.

Технологии.
Основными технологиями являются python с его фреймворком Flask. Также используются дополнения к flask, а именно flask-login и flask-sqlalchemy для организации системы входа пользователей и доступа к базе данных. Интерфейс построен на базе bootstrap заготовок. В качестве обслуживающих подключение пользователей планируются использовать следующие технологии: uwsgi и nginx. А для обеспечения изоляции сервиса от других процессов планируется запуск через docker или связку docker и docker-compose.

Запуск.
Для запуска клиента необходим только браузер. Актуальный список необходимых для запуска технологий, а также инструкции по запуску, можно будет найти в README файле репозитория.
